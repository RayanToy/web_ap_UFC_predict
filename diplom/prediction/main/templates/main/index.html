Прогноз дерева решений

Дерево решений — это классификатор, построенный на правилах «если, то», организованных в виде иерархической древовидной конструкции. Дерево решений основано на процессе рекурсивного разделения начального набора объектов на подмножества, связанные с предопределенными классами. Разделение выполняется с помощью правил принятия решений, где значения атрибутов проверяются на основе заданного условия. Строится на основе обучения с учителем. Набор наблюдений используется в качестве обучающего набора данных, для которого предопределена метка класса. Конструкция дерева решений представляется из объектов двух типов: узлов и листьев. Узлы содержат правила принятия решений и подмножества наблюдений, которые им удовлетворяют. Листья содержат классифицированные наблюдения из дерева: каждый лист является одним из классов, а объекту, развернутому в листе, присваивается соответствующая метка класса.
